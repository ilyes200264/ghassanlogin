## 🚀 Déploiement

Le site est configuré pour être déployé sur Vercel avec :
- Variables d'environnement Supabase
- Build automatique depuis GitHub
- Redirection automatique après authentification

## 🛠️ Technologies

- **Next.js 14** - Framework React
- **Supabase** - Authentification et base de données
- **Tailwind CSS** - Styling et design
- **Lucide React** - Icônes
- **TypeScript** - Type safety
- **Vercel** - Déploiement

## 📱 Responsive Design

Le site s'adapte parfaitement à tous les écrans :
- **Mobile** : Interface optimisée pour les petits écrans
- **Tablet** : Layout adaptatif avec grilles flexibles
- **Desktop** : Interface complète avec toutes les fonctionnalités
